in the following video you will learn about code monkeys skill mode challenges skill mode challenges are additional challenges which are similar to the coding adventure challenges however their purpose is to provide students with more practice rather than teach them new concepts to access the skill mode challenges go to your courses page from the my courses section you can access skill mode challenges each correlating with the relevant coding adventure course click on the course and then click on start coding you can also click on the green map on the righthand side of the screen to toggle between coding adventure challenges and skill mode challenges some skill mode challenges will be locked based on where you are in the course to unlock the rest of skill mode continue solving coding adventure challenges for teachers all skill mode challenges will be open for more information on skill mode click on the link in the video's description or visit code monkeys Help Center thank you for watching this video on code monkeys skill mode challenges
